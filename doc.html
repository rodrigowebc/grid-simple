Grid Simple

Um sistema de grids simples desenvolvido por Brasileiros!

Documentação:

Para usar esse o Grid Simple é MUITO FÁCIL! Primeiramente, você deve baixar e copiar os arquivos para pasta raiz do seu site. Depois disso é só chamar o arquivo grid-simple.css no seu HTML!

<link rel="stylesheet" href="css/grid-simple.css" type="text/css">

Agora, você precisa criar algumas div's. Veja abaixo como:

<div class="container">
   <div class="row">
      <div class="grid-12"> ...conteúdo... </div> 
   </div> 
</div>
