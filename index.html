Grid Simple

Um sistema de grids simples desenvolvido por Brasileiros!

Veja como é fácil:

Baixe o arquivo .rar neste link ou clone no seu repositório via GitHub, copie os arquivos para pasta raiz do seu site e chame o arquivo grid-simple.css no seu HTML! Simples assim :)

<link rel="stylesheet" href="css/grid-simple.css" type="text/css">
